Design patterns for modern JavaScript

The IE11 winter is over! While we were hibernating, JavaScript evolved dramatically — yet most of our design patterns haven’t. Developers (yes, you) — it’s time to refresh how we author our “everyday code” and leverage these new tools! This talk will discuss new patterns for creating a global store, memoizing instance data, emitting events, parsing strings, and observing DOM elements. The future of JavaScript is already here — we just need to learn how to use it.
